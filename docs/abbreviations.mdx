# Abbreviations

There are so many abbreviations in the IT industry. And you will even find the
same abbreviations with different meanings. So you always have to be aware of
the context in which you hear or read the abbreviation.

## C

- CD - Continuous Delivery or Continuous Deployment
- CI - Continuous Integration
- CI/CD - Continuous Integration (CI) / Continuous Delivery or Continuous Deployment (CD)

## D

- DORA - DevOps Research and Assessment
- Docs-as-Code - Documentation as Code

## I

- IaC - Infrastructure as Code
